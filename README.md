# TZVR-IT

## О проекте

Это тестовое задание. Данное простое консольное приложение на C++ реализует сортировку массива различными алгоритмами в отдельных потоках.

## Реализованные алгоритмы сортировки

- Пузырьком (Bubble Sort)
- Выбором (Selection Sort)
- Вставками (Insertion Sort)
- Слиянием (Merge Sort)
- Быстрая (Quick Sort)

## Особенности

- Каждый алгоритм работает в отдельном потоке.
- После завершения каждой сортировки результат выводится в консоль.
- Добавлены автотесты с разными тестовыми данными для проверки граничных условий.

## Запуск проекта

### Через Visual Studio

1. Откройте Visual Studio.
2. Нажмите **"Клонировать репозиторий"**.
3. Вставьте ссылку на репозиторий GitHub.
4. Дождитесь загрузки проекта.
5. Нажмите **"Запуск"** (или F5) для старта приложения.

### Через командную строку (MSVC)

1. Откройте **x64 Native Tools Command Prompt for VS**.
2. Перейдите в папку с проектом.
3. Скомпилируйте проект командой:
   ```sh
   cl /EHsc /std:c++17 /Fe:sorting_app.exe TZVR-IT.cpp ThreadSort.cpp
   ```
4. Запустите приложение:
   ```sh
   sorting_app.exe
   ```

## Основные файлы

- `TZVR-IT.cpp` - Главный файл проекта.
- `ThreadSort.cpp` - Реализация алгоритмов сортировки.
